Sandbox

The Sandbox application is an application used to test the dyn4j project.

Sandbox takes the place of an older test application, TestBed, and offers a much richer set of features for testing, mainly the ability to create a scene without programming. These scenes can be saved in a human readable XML document which can be loaded later. It contains all the features of the old application and more.
