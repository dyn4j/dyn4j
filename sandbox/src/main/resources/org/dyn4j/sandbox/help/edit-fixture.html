Editing Fixtures

Fixtures can be edited, however, the shape fixture's shape cannot be. If you need to change the shape of a fixture, you must remove the fixture and create a new one with the proper shape.

You can edit the filter, sensor, transform, and other properties about the fixture by clicked the Edit option on the Fixture context menu.



The sensor property allows the fixture to be detected for collisions but not resolved.

The density property allows the fixture to be more/less dense than other fixtures on the body.

The filter property allows the fixture to be filter to only collide with other fixtures in the same category.

The friction property allows the fixture to have a coefficient of friction (slowing). When two fixtures collide their friction properties are mixed using a default mixing function.

The restitution property allows the fixture to have a coefficient of restitution (bounce). When two fixtures collide their restitution properties are mixed using a default mixing function.
