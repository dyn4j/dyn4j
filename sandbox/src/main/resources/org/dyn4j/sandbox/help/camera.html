Camera Control

You can move around the simulation by clicking and holding the Left Mouse Button any where in the simulation area and dragging the mouse.

You can zoom in and out using the Mouse Scroll Wheel or the Zoom In and Zoom Out toolbar buttons.

The To Origin toolbar button will re-center the camera on the origin.



The dyn4j library works in the MKS (meters-kilogram-seconds) measuring system. Because of this, all length values must be converted into pixels using a scale factor.

In the bottom left corner of the Simulation Window is a scale to help measure bodies. The scale shows a fixed size line and gives the number of meters it covers. Above that is the number of pixels per meter. The scale factor is basically the zoom amount.
