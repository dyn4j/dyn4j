Preferences

The Sandbox application allows you to decide what is rendered. This aids in debugging issues with the library or with a specific simulation. In addition, the rendering options also allow a number of color choices to better aid in visual recognition.

Preferences are stored across simulations but are not stored to a file by the application. Because of this, the preferences will not be saved from one run of the Sandbox to another.

You can change the preferences using the Preferences Quick Toolbar or by clicking on Window → Preferences. You can hover over any of the Quick Preferences Toolbar buttons to see a short description of each option.


The preferences toolbar allows you to simply toggle rendering. To change the colors of the rendered objects open the Preferences Dialog using the Window menu.


To toggle rendering of any object, click the checkbox. To change the color of the rendered objects click the colored square next to the check box. When you are done, click the Apply button at the bottom of the dialog.
