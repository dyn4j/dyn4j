Export

In addition to being able to save a simulation to an XML document, the simulation can be exported to a Java source file to be added into your project. This allows the Sandbox to be used as a starting point for a level editor.

You can export a simulation by clicking the Export → Java option in the File menu.


The exported Java source file will contain one static method that will configure a given world object. This class will add comments for the names of all the objects and will attempt to use the name of the simulation as the name of the class.
