New, Open, & Save

The Sandbox application starts with a blank simulation already created and ready to be setup. To create a new simulation, use the New option in the File menu.


Simulations can also be saved to a file (XML) using the Save As... option in the File menu.
  • Sending this saved simulation file with a bug report is the best way to get bugs resolved for dyn4j.

You can open a previously saved (XML) simulation file using the Open option in the File menu. The Open option can open any version of a Sandbox simulation.

Creating a new simulation or opening an existing simulation will clear all snapshots and reset the current world settings ().
