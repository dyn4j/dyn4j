Adding Bodies & Fixtures

Creating bodies is the start to setting up a simulation. Bodies represent the objects in the simulation.

To add a body Right Click or Command Click on the Bodies folder node in the Simulation Tree on the left:



This will bring up a new dialog window where you can setup the new body. Hover over any of the i icons to see help information about the different settings. Usually you will only change the name of the body and maybe its mass type.
  • If you are creating the ground body, choose a mass type of Infinite.
  • All the options are available later when editing a body.
  • You can use the Transform tab to manually move a body to a specific position.




When you are done setting up the body click the Add button at the bottom of the dialog. You will now see the body in the Simulation Tree. Next we need to specify what the body looks like, or its shape. We do this by adding fixtures to the body. A body can have multiple fixtures. To add a fixture Right Click or Command Click the body you created and choose the fixture type you would like to create.
  • A convex hull fixture will create a convex shape from a point cloud.
  • A non-convex polygon can be decomposed into a number of fixtures.




For simplicity we will create a Rectangle fixture. This will bring up a new dialog where you can set the rectangle's properties. You can also setup the fixture settings using the Fixture tab. The fixture settings include density, filters, and coefficients.
  • The local transform tab allows you to move the new fixture locally to the body. This is useful if the body will have more than one fixture. Be aware that this modifies the vertices of the fixture directly.
  • The shape of the fixture cannot be edited later.




When you are done setting up the new fixture click the Add button at the bottom of the dialog. Will now see the new fixture on the body in the Simulation Tree. You will also see the body (its fixtures) rendered in the Simulation window on the right.
